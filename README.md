#  ThymeTraque

Simple time tracking app.

## Supported Platforms

iOS

## Running the app

- Open `ThymeTraque.xcodeproj` in Xcode.
- Choose `ThymeTraque` schem and a suitable destination.
- Click `Run` or `Cmd+R`.

## Running the tests

- Open `ThumeTraque.xcodeproj` in Xcode.
- Click `Product` > `Test` or `Cmd+U`.
